1. Estrutura base do documento
Todo arquivo HTML começa com a mesma estrutura base. O <!DOCTYPE html> é sempre a primeira linha e avisa ao navegador que o arquivo é HTML5. Em seguida vem a tag <html>, que é a raiz do documento e envolve tudo — ela recebe o atributo lang="pt-br" para indicar o idioma da página.
Dentro do <html> existem dois grandes blocos: o <head> e o <body>. O <head> é a área invisível, onde ficam as configurações da página. O <body> é onde fica tudo que o usuário vê na tela.

2. O <head> e suas tags
Dentro do <head> ficam os metadados, ou seja, informações sobre a página que não aparecem diretamente na tela.
A tag <meta charset="UTF-8"> define a codificação de texto, evitando problemas com acentos e caracteres especiais do português. A tag <meta name="viewport" content="width=device-width, initial-scale=1.0"> garante que a página se adapte corretamente ao tamanho de celulares e tablets.
A tag <title> define o nome que aparece na aba do navegador. É importante para identificação da página e também para o Google.

3. Organização semântica do <body>
O <body> é dividido em seções com significado próprio, chamadas de tags semânticas. Isso ajuda tanto o navegador quanto leitores de tela a entenderem a estrutura da página.
O <header> representa o cabeçalho visual da página, geralmente contendo o título principal e o menu de navegação. O <main> envolve o conteúdo principal da página e deve existir apenas um por página. O <footer> representa o rodapé, onde geralmente ficam informações de copyright, contato e links úteis.
Dentro do <main>, a tag <section> divide o conteúdo em blocos temáticos, como capítulos de um livro. Cada seção costuma ter seu próprio título.

4. Títulos e parágrafos
Os títulos vão de <h1> até <h6>, em ordem decrescente de importância. O <h1> é o título principal da página, o maior e mais importante, e deve existir apenas um por página. O <h2> é usado para subtítulos de seções, o <h3> para sub-subtítulos, e assim por diante até o <h6>.
O <p> cria um parágrafo de texto. O navegador adiciona automaticamente um espaço acima e abaixo de cada parágrafo.

5. Formatação de texto
Para formatar o texto existem várias tags específicas. O <strong> deixa o texto em negrito e indica que ele é importante, tendo significado semântico. O <b> também deixa em negrito, mas é apenas visual, sem indicar importância. O recomendado é sempre usar <strong>.
O <em> deixa o texto em itálico e indica ênfase. A tag <br> cria uma quebra de linha e não precisa de tag de fechamento.
A tag <abbr title="..."> define uma abreviação. O texto dentro do atributo title aparece como uma dica quando o usuário passa o mouse em cima da palavra.
Os comentários são escritos com <!-- texto --> e não aparecem na página, servindo apenas como anotações para o programador.

6. Links e imagens
O <a href="..."> cria um link clicável. O atributo href define o endereço de destino. O atributo target="_blank" faz o link abrir em uma nova aba do navegador.
A tag <img> exibe uma imagem e não tem tag de fechamento. Seus principais atributos são o src, que define o caminho ou link da imagem, o alt, que é o texto alternativo exibido caso a imagem não carregue e lido por leitores de tela, e o width, que define a largura em pixels. O <figcaption> é usado para adicionar uma legenda abaixo de uma imagem.

7. Formulários
O <form> agrupa todos os campos de um formulário. Dentro dele ficam os inputs, que são os campos onde o usuário interage.
A tag <input> é a mais versátil do HTML. Seu comportamento muda completamente dependendo do atributo type. Os principais tipos que você aprendeu são:

type="text" — campo de texto livre
type="password" — campo de texto que esconde o que é digitado
type="email" — valida automaticamente se o texto parece um email
type="number" — aceita apenas números, com min e max opcionais
type="url" — valida se o texto parece uma URL
type="radio" — botão de seleção circular, permite escolher apenas uma opção do grupo
type="checkbox" — caixinha de marcação, permite marcar várias opções ao mesmo tempo
type="file" — botão para upload de arquivos, com accept para filtrar o tipo
type="date" — abre um calendário para escolher uma data
type="time" — abre um seletor de horário
type="month" — abre um seletor de mês e ano
type="week" — abre um seletor de semana do ano
type="range" — cria uma barra deslizante (slider)
type="color" — abre um seletor de cores
type="submit" — botão que envia o formulário

O atributo placeholder exibe um texto cinza dentro do campo quando ele está vazio, dando uma dica ao usuário. O id é o identificador único do elemento. O name nomeia o campo para quando os dados forem enviados. O value define o valor do campo.
A tag <label> cria um texto clicável associado a um input. O atributo for do label precisa ter o mesmo valor que o id do input correspondente, fazendo com que clicar no texto também ative o campo.
O <textarea> é um campo de texto para respostas longas. Os atributos rows e cols definem o número de linhas e colunas visíveis.
O <select> cria um menu suspenso. Dentro dele ficam as tags <option>, onde o value é o valor enviado e o texto entre as tags é o que o usuário vê na tela.
O <fieldset> agrupa campos relacionados dentro de um formulário, desenhando uma borda ao redor. A tag <legend> define o título desse agrupamento, aparecendo na borda superior.

8. Tabelas
As tabelas são criadas com a tag <table>. Elas são divididas em três partes: <thead> para o cabeçalho, <tbody> para o corpo com os dados e <tfoot> para o rodapé.
Dentro dessas seções, <tr> cria uma linha. Dentro das linhas, <th> cria uma célula de cabeçalho (negrito e centralizado automaticamente) e <td> cria uma célula normal de dados. O atributo colspan faz uma célula ocupar o espaço de mais de uma coluna.

9. Listas
O <ol> cria uma lista ordenada, onde os itens são numerados automaticamente. O <ul> cria uma lista não ordenada, onde os itens aparecem com bolinhas. Em ambos os casos, cada item da lista é marcado com a tag <li>.

10. Conteúdo expansível e outros elementos
O <details> cria um bloco que pode ser aberto e fechado pelo usuário com um clique. A tag <summary> define o texto clicável que funciona como título desse bloco.
O <span> é um container inline, ou seja, não quebra linha. É usado para envolver um trecho de texto para aplicar estilos ou capturar valores com JavaScript.
As entidades HTML são códigos especiais para representar caracteres que não podem ser digitados diretamente no código. Por exemplo, &copy; representa o símbolo © de copyright.
