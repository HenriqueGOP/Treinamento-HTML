0. O que é CSS e para que serve?

Se o HTML é o esqueleto de uma página web (estrutura e conteúdo), o CSS é a roupa e a maquiagem — ele define como tudo vai aparecer na tela.

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). Com ele você controla cores, tamanhos, fontes, espaçamentos, posicionamento, animações e muito mais.

Como usar o CSS?

Existem três formas de aplicar CSS em um HTML:

1. CSS externo (recomendado) — você cria um arquivo separado com extensão .css e o linkeia no HTML dentro do <head>:

<link rel="stylesheet" href="estilo.css">

2. CSS interno — você escreve o CSS dentro de uma tag <style> no <head> do próprio HTML:

<style> body { background-color: blue; } </style>

3. CSS inline — você escreve o estilo diretamente na tag HTML com o atributo style. Pouco recomendado:

<p style="color: red;">Texto vermelho</p>

Sintaxe básica do CSS

O CSS é escrito em blocos chamados regras. Cada regra tem um seletor (quem vai ser estilizado) e uma ou mais declarações (o que vai mudar):

seletor { propriedade: valor; outra-propriedade: outro-valor; }

Os comentários no CSS são escritos com /* texto */ e não aparecem na página.

Texto vermelho criado com CSS!

1. @import — Importando fontes externas

@import url('https://fonts.googleapis.com/css2?family=Trispace&display=swap');

O @import importa um recurso externo para dentro do CSS — nesse caso, uma fonte do Google Fonts. Sem isso, você só pode usar as fontes já instaladas no computador do usuário.

Depois de importar, você usa a fonte pelo nome dentro da propriedade font-family:

body { font-family: "Trispace", sans-serif; }

O sans-serif após a vírgula é a fonte de fallback (reserva): se a Trispace não carregar, o navegador usa qualquer fonte sans-serif disponível.

2. Seletor Universal *

* { margin: 0; padding: 0; box-sizing: border-box; }

O asterisco * seleciona todos os elementos da página de uma só vez. É o seletor mais abrangente do CSS.

margin e padding

Todos os elementos HTML têm margens e preenchimentos internos que o navegador coloca por padrão. Isso pode causar espaços indesejados. Zerando com margin: 0 e padding: 0, você parte do zero e controla tudo manualmente.

  • margin — espaço do lado de fora do elemento (entre ele e os vizinhos)
  • padding — espaço do lado de dentro do elemento (entre a borda e o conteúdo)

box-sizing: border-box

Por padrão, quando você define width: 200px, o padding e a borda são somados por cima, tornando o elemento maior que 200px. Com box-sizing: border-box, o padding e a borda já são incluídos dentro do tamanho definido — o que torna os cálculos muito mais previsíveis.

💡 Dica: Esse bloco * { margin: 0; padding: 0; box-sizing: border-box; } é chamado de CSS Reset e praticamente todo projeto profissional começa com ele.
box-sizing: border-box
O padding verde (interno) e a borda laranja não aumentam o tamanho final desta caixa. Tudo se encaixa perfeitamente.

3. Propriedades do body

body { background-color: rgb(185, 89, 230); font-family: "Trispace", sans-serif; color: #1a1a1a; line-height: 1.6; }

Cores: rgb() e #hex

No CSS existem várias formas de escrever uma cor:

  • rgb(185, 89, 230) — define a cor pelos canais Red, Green e Blue, cada um de 0 a 255
  • #1a1a1a — formato hexadecimal, muito comum. Cada par de letras/números representa R, G e B em base 16
  • white, pink, blue — nomes de cores em inglês, direto no código

line-height

Define a altura de cada linha de texto. O valor 1.6 significa 1,6× o tamanho da fonte atual. Aumentar o line-height deixa o texto mais arejado e fácil de ler.

4. Espaçamento e Sombra

header { padding: 3rem 1rem; text-align: center; box-shadow: 0 4px 10px rgba(0,0,0,0.2); }

padding com dois valores

Quando você usa dois valores no padding, o primeiro se aplica ao topo/baixo e o segundo à esquerda/direita:

padding: 3rem 1rem; /* topo/baixo: 3rem | esquerda/direita: 1rem */

rem — unidade de medida

rem é uma unidade relativa ao tamanho de fonte padrão do navegador (normalmente 16px). Assim, 1rem = 16px, 2rem = 32px, etc. É mais flexível que usar pixels fixos.

box-shadow

Adiciona uma sombra ao elemento. Os valores são, em ordem:

box-shadow: X Y desfoque cor; /* Exemplo: */ box-shadow: 0 4px 10px rgba(0,0,0,0.2);
  • 0 — deslocamento horizontal (0 = sem deslocamento)
  • 4px — deslocamento vertical (para baixo)
  • 10px — quanto a sombra vai se desfocar (blur)
  • rgba(0,0,0,0.2) — cor da sombra (explicado abaixo)

rgba() — cor com transparência

O rgba() é como o rgb(), mas com um quarto valor: a opacidade (alpha), de 0 a 1.

rgba(0, 0, 0, 0.2) /* preto com 20% de opacidade */ rgba(255, 255, 255, 0.5) /* branco com 50% de opacidade */
Esta caixa tem padding (espaço interno) e um box-shadow escuro com transparência, criando a ilusão de que está flutuando.

5. Flexbox — Organizando elementos

main { max-width: 900px; margin: 0 auto; display: flex; flex-direction: column; gap: 3rem; }

max-width

Limita a largura máxima do elemento. O conteúdo nunca vai ser mais largo que 900px, mas pode ser menor em telas pequenas.

margin: 0 auto

Com o max-width definido, usar margin: 0 auto centraliza o elemento horizontalmente na página. O auto divide o espaço restante igualmente dos dois lados.

display: flex

Ativa o Flexbox, um sistema de layout moderno do CSS. Quando você coloca display: flex em um elemento, os filhos diretos dele se organizam de forma flexível.

flex-direction

Define a direção em que os filhos serão organizados:

  • row (padrão) — lado a lado, em linha horizontal
  • column — um embaixo do outro, em coluna vertical

gap

Define o espaço entre os filhos do elemento flex. É muito mais prático do que ficar colocando margin em cada filho manualmente.

6. Bordas

section { border-radius: 12px; border-left: 6px solid #4b0082; }

border-radius

Arredonda os cantos do elemento. Quanto maior o valor, mais arredondado fica. Com border-radius: 50% você transforma um quadrado em um círculo perfeito.

border-left (e variações)

Aplica borda apenas em um lado específico. O valor é escrito em três partes: espessura, estilo e cor.

border-left: 6px solid #4b0082; /* espessura | estilo | cor */ /* Outros lados: */ border-top: ...; border-right: ...; border-bottom: ...; /* border: (aplica nos 4 lados ao mesmo tempo) */
Elemento com os cantos arredondados (border-radius) e uma borda grossa roxa apenas no lado esquerdo (border-left).

7. Seletores de Atributo input[type="..."]

input[type="radio"] { border-radius: 50%; } input[type="checkbox"] { border-radius: 4px; }

Você pode selecionar elementos com base nos seus atributos HTML. A sintaxe é elemento[atributo="valor"]. Isso permite estilizar apenas os inputs do tipo radio, ou apenas os do tipo checkbox, por exemplo — sem precisar adicionar classes no HTML.

8. Estilizando inputs customizados

input[type="radio"], input[type="checkbox"] { appearance: none; -webkit-appearance: none; width: 18px; height: 18px; cursor: pointer; position: relative; vertical-align: middle; }

appearance: none

Remove o visual padrão que o sistema operacional aplica nos inputs (aquele estilo cinza padrão de rádio/checkbox). Sem isso, o navegador sobrescreve seu CSS. O -webkit-appearance é o mesmo, mas para compatibilidade com navegadores baseados em WebKit (como o Safari).

width e height

Definem a largura e a altura do elemento, respectivamente. Podem receber valores em px, rem, %, entre outros.

cursor

Define como o cursor do mouse aparece ao passar sobre o elemento. pointer exibe a "mãozinha", indicando que é clicável.

position: relative

Ativa o sistema de posicionamento do elemento. Com relative, o elemento continua no fluxo normal da página, mas agora serve como referência de posição para filhos com position: absolute.

vertical-align: middle

Alinha o elemento verticalmente ao centro em relação ao texto ao seu lado. Útil para inputs que ficam ao lado de labels.

9. Pseudo-classes: :checked, :focus, :hover

Pseudo-classes são palavras-chave adicionadas a um seletor que aplicam estilo apenas quando o elemento está em um estado específico.

input[type="radio"]:checked { background-color: pink; } input:focus { outline: none; border-color: pink; } input[type="submit"]:hover { background-color: #6b2a60; }
  • :checked — ativo quando o radio/checkbox está marcado
  • :focus — ativo quando o elemento está selecionado (clicado ou com Tab). O outline: none remove o contorno azul padrão do navegador
  • :hover — ativo quando o mouse passa por cima do elemento

10. Pseudo-elemento ::after

input[type="checkbox"]:checked::after { content: "✓"; position: absolute; color: #4b0082; font-size: 14px; left: 2px; top: -3px; }

::after cria um elemento visual "fantasma" que aparece depois do conteúdo do elemento, sem precisar adicionar nada no HTML. O par é o ::before, que aparece antes. A propriedade content é obrigatória — é onde você define o que vai aparecer.

position: absolute

Remove o elemento do fluxo normal da página e o posiciona em relação ao seu pai com position: relative. Aí você usa top, left, right e bottom para definir a posição exata.

font-size

Define o tamanho da fonte do elemento. Pode ser em px, rem, em, entre outros.

11. Animações simples: transition e transform

input[type="submit"] { transition: 0.3s; text-transform: uppercase; } input[type="submit"]:hover { transform: translateY(-2px); }

transition

Faz com que as mudanças de estilo aconteçam de forma suave (animada) ao invés de instantânea. O valor 0.3s significa que a transição leva 0,3 segundos. Você pode especificar qual propriedade animar ou usar all para todas.

transform: translateY()

Move o elemento sem afetar o espaço que ele ocupa na página. translateY(-2px) sobe o elemento 2 pixels. Combinado com o transition, cria o efeito de "flutuar" ao passar o mouse.

text-transform

Altera a capitalização do texto sem mudar o HTML:

  • uppercase — tudo em MAIÚSCULAS
  • lowercase — tudo em minúsculas
  • capitalize — Primeira letra de cada palavra maiúscula

12. Estilizando tabelas

table { width: 100%; border-collapse: collapse; overflow: hidden; }

width: 100%

Faz o elemento ocupar 100% da largura do seu elemento pai.

border-collapse: collapse

Por padrão, cada célula da tabela tem sua própria borda, o que resulta em bordas duplicadas entre células. border-collapse: collapse junta essas bordas em uma só.

overflow: hidden

Esconde qualquer conteúdo que ultrapasse os limites do elemento. Aqui é usado para que o border-radius da tabela funcione nos cantos (sem isso, as células dos cantos "vazariam" para fora da borda arredondada).

13. text-decoration

footer a { text-decoration: none; } footer a:hover { text-decoration: underline; }

Controla decorações no texto:

  • none — remove o sublinhado padrão dos links
  • underline — adiciona sublinhado
  • line-through — adiciona tachado (riscado)

Note também o seletor footer a: isso seleciona apenas as tags <a> que estão dentro de um <footer>. Você pode combinar seletores assim para ser mais específico.

14. Classes CSS: .nome-da-classe

.form-group { display: flex; flex-direction: column; gap: 0.4rem; margin-bottom: 1.2rem; } .btn-submit { align-self: flex-start; padding: 10px 30px !important; }

As classes são a principal forma de aplicar CSS de forma reutilizável. No CSS, uma classe é definida com um ponto antes do nome (.nome). No HTML, você aplica a classe com o atributo class="nome":

<div class="form-group">...</div>

Um elemento pode ter várias classes separadas por espaço: class="form-group btn-submit".

align-self

Quando os filhos estão dentro de um display: flex, o align-self permite que um filho específico quebre o alinhamento padrão do grupo. flex-start alinha o elemento no início do eixo transversal (à esquerda, quando a direção é coluna).

!important

Força aquela declaração a ter prioridade máxima, sobrescrevendo qualquer outra regra que tente definir a mesma propriedade naquele elemento. Deve ser usado com moderação — em projetos grandes, abusar do !important torna o CSS difícil de manter.

15. Herança: inherit e font-weight

label { font-weight: 500; } input { font-family: inherit; }

font-weight

Controla a espessura do texto. Os valores mais comuns são:

  • 400 — peso normal (equivale a normal)
  • 500 — meio-negrito
  • 700 — negrito (equivale a bold)
  • 900 — negrito extra-pesado

inherit

Força o elemento a herdar o valor da propriedade do seu elemento pai. Inputs, por padrão, não herdam a fonte do body — então font-family: inherit força o input a usar a mesma fonte definida no body.
